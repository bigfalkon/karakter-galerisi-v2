🔥 Firebase Migration Tool

⚠️ Başlamadan önce: Yeni Firebase projesinde Firestore kurallarını geçici olarak açık yap:
Firebase Console → Firestore → Rules → aşağıdaki kodu yapıştır → Publish

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Migration bittikten sonra kuralları eski haline getirmeyi unutma!
